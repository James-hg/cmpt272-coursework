Basics of Web Design

By Felke-Morris

Overview

This textbook provides a foundational approach to modern web design, covering HTML5, CSS3, and responsive principles. Felke-Morris guides students through the process of creating accessible and well-structured websites with practical exercises. It is an ideal resource for beginners looking to build a strong technical and creative base in web development.
